# Guía de Instalación - Robot Delivery Simulator

## 🎯 Instalación para Usuarios Finales

1. **Descarga** el archivo ZIP desde GitHub
2. **Extrae** el contenido en una carpeta de tu elección
3. **Ejecuta** el archivo `StartGame.bat`
4. **Conecta** tu mando PS5 (opcional)
5. **¡Juega!**

## 🔧 Solución de Problemas

### ❌ Error: "Falta VCRUNTIME140.dll"
**Solución:** Instala [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### ❌ Error: "No se encuentra SFML.dll"
**Solución:** Asegúrate de que todas las DLLs estén en la carpeta `libraries/`

### ❌ El mando PS5 no funciona
**Solución:**
- Conecta el mando ANTES de ejecutar el juego
- Usa cable USB para mejor compatibilidad

## 📋 Requisitos del Sistema

- Windows 10 o superior (64-bit)
- 2 GB RAM mínimo
- Tarjeta gráfica con OpenGL 2.1+
- 100 MB espacio libre
