# 🚗 Robot Delivery Simulator - Miami

Simulador de vehículo de delivery con controles de mando PS5, ambientado en las calles de Miami.

## 🎮 Características

- Control realista con mando PS5
- 3 velocidades + reversa con físicas realistas  
- Mapa real de Miami como entorno
- Sistema de tacómetro en tiempo real
- Gráficos suaves a 60 FPS

## 🕹️ Controles PS5

- **L1 + R2**: Acelerar hacia adelante
- **L1 + L2**: Reversa
- **Stick Izquierdo**: Girar vehículo
- **Triángulo ▲**: Subir cambio (1 → 2 → 3)
- **X**: Bajar cambio (3 → 2 → 1)
- **ESC**: Salir del juego

## 🚀 Instalación Rápida

1. **Descarga** la última versión
2. **Extrae** el archivo ZIP en cualquier carpeta
3. **Ejecuta** `StartGame.bat`
4. **¡Conecta tu mando PS5 y conduce!**

### Requisitos del Sistema
- Windows 10 o superior
- 100 MB de espacio libre
- Mando PS5 (opcional)

## 🛠️ Tecnologías Utilizadas

- **C++17** - Lenguaje de programación
- **SFML 2.6** - Multimedia y gráficos

## 📞 Soporte

Si tienes problemas para ejecutar el simulador:
1. Revisa el archivo INSTALL.md
2. Verifica que tengas instalados los [Visual C++ Redistributables](https://aka.ms/vs/17/release/vc_redist.x64.exe)
